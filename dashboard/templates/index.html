Pipeline Setup Wizard

Configure the V2F pipeline in 6 steps. All values are written to config/pipeline_config.yaml when you click Save.

1Adapter type

Picks how the VCF is loaded and which ACMG criteria are eligible.

2VCF input path

Local path or gs:// URI. You can also drop a file in the Upload tab — it will populate this field.

3Reference genome

4VEP annotation mode

Stage 2 uses VEP for consequence prediction. skip bypasses VEP if your VCF is already annotated.

5Reference databases

BigQuery tables for ClinVar pathogenicity and gnomAD frequencies. Defaults below are the public datasets.

6FHIR phenotype integration (optional)

Pulls patient phenotype from a FHIR Condition table to score VUSes against the patient's actual disease profile.
